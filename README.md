# Amazon Homepage Clone

A simple frontend project that recreates the look and feel of the Amazon homepage using only **HTML** and **CSS**. This project focuses on practicing webpage structure, layouts, styling, and creating a user interface similar to a real-world e-commerce website.

## 📌 Project Overview

This project is a static clone of the Amazon homepage built from scratch without using any frontend frameworks or libraries. The goal of this project was to improve my understanding of:

- HTML page structure
- CSS styling and layouts
- Flexbox and positioning
- Navigation bar design
- Product card layouts
- Creating responsive webpage sections

This project is created for learning and educational purposes.

## 🚀 Features

- Amazon-style navigation bar
- Search bar UI
- Header and menu sections
- Hero/banner section
- Product category sections
- Product cards with images
- Footer section
- Amazon-inspired styling and layout

## 🛠️ Technologies Used

- **HTML5** - Used for creating the structure of the webpage
- **CSS3** - Used for styling, layouts, and visual design
